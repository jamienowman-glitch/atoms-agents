## Colour palette

- Header text: `accordion_header_text` token referencing primary text on black base.
- Body text: `accordion_body_text` token aligned to secondary text on black base.
- Chevron: `accordion_chevron_colour` token; inherits header text by default.
- Divider/stroke: `accordion_divider_stroke` token; optional to separate stacked items.
- Background: `accordion_header_background` and `accordion_body_background` tokens default to transparent/parent surface; no hardcoded hex.
