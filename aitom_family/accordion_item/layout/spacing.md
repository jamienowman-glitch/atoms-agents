## Spacing

- Vertical spacing between stacked items via `accordion_item_gap` token; default matches theme spacing scale.
- Internal gap between header text and chevron via `accordion_header_content_gap`.
- Body spacing to surrounding content via `accordion_body_margin_top` token; collapses when closed.
- Mobile overrides: `accordion_header_padding_mobile`, `accordion_body_padding_mobile`, and `accordion_item_gap_mobile` allow tighter stacks on small screens without breaking 24-column alignment.
- Spacing above/below the entire atom controlled by `accordion_block_margin_top/bottom` tokens instead of hardcoded margins.
