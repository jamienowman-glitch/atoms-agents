## Axis tokens

- Header axes: `accordion_header_wght`, `accordion_header_wdth`, `accordion_header_opsz`, `accordion_header_slnt` (optional), `accordion_header_tracking`, `accordion_header_line_height`.
- Body axes: `accordion_body_wght`, `accordion_body_wdth`, `accordion_body_opsz`, `accordion_body_slnt` (optional), `accordion_body_tracking`, `accordion_body_line_height`.
- Density adjustments use `accordion_header_line_height_compact/comfortable` and `accordion_body_line_height_compact/comfortable` tokens instead of fixed pixel overrides.
- All axes map to Roboto Flex via the theme font helper; no raw numbers hardcoded here.
