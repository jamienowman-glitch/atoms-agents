## Exposed tokens

- `spacer_size_default` (enum xs–xl) – base height selection tied to spacing scale.
- `spacer_size_mobile`, `spacer_size_tablet`, `spacer_size_desktop` – optional per-breakpoint overrides.
- `spacer_density_modifier` (enum none|compact|comfortable) – adjusts height up/down one scale step.
- `spacer_min_height`, `spacer_max_height` – clamps to keep spacing within safe bounds.
- `spacer_debug_outline_enabled` and `spacer_debug_outline_stroke` – optional debug visibility controls without changing layout.
