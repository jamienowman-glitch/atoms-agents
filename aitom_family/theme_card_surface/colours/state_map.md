## State map

- Default: `card_surface_dark`, `card_stroke`.
- Hover: `card_surface_hover`, `card_stroke_hover`, optional `card_glow_colour`.
- Pressed: `card_surface_pressed`, `card_stroke`.
- Focus: `card_surface_dark`, `card_stroke`, `card_focus_ring`.
- Selected: `card_surface_selected`, `card_stroke_selected`, maintain focus ring if focused.
- Disabled: `card_surface_disabled`, `card_stroke_disabled`, reduced opacity token.
- Provide equivalents for light variant tokens.
