## Default values and ranges

- Surfaces: dark=#0b0b0b, hover=#111, pressed=#0a0a0a, selected=#141414, disabled=#0b0b0b with opacity 0.5; light variant optional (surface=#fff).
- Strokes: default=#fff, hover=#fff, selected=#fff, disabled=rgba(255,255,255,0.4).
- Focus ring: colour=#fff, width=2px.
- Radius scale: default 12px; range 8–20px.
- Padding: compact 12–16px, comfortable 16–24px.
- Elevation/glow: default 0, hover glow opacity 0.12, pressed 0.06, selected 0.16; blur 12–24px.
- Badge: position top-right inset respecting radius; icon size 14–16px; background uses stroke token by default.
- Density default: comfortable; tokens override per template.
