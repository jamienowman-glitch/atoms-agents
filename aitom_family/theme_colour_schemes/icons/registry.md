## Icons registry

- Default picker uses text + swatch; optional icons:
  - `checkmark` for selected scheme indicator (use existing monochrome check icon).
  - `contrast_warning` icon for low-contrast variants (optional).
- Tokens:
  - `scheme_icon_size`
  - `scheme_icon_colour` (paired with scheme text colour tokens; invert on light swatches)
  - `scheme_icon_position` (start/end or overlay on swatch)
- If icons not used, keep tokens dormant but documented to allow future swap.
