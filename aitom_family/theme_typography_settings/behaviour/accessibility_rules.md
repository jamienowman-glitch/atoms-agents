## Accessibility rules

- Support user font-size scaling: allow multiplier token with clamped min/max; reflow layout after update.
- Maintain line-length guidance: suggest max 72ch for body, 45–60ch ideal; expose token for clamps.
- Respect prefers-reduced-motion: disable animated size changes; apply instantly.
- Ensure minimum text sizes for legibility: body ≥ 14px, caption ≥ 12px, buttons ≥ 14px with 16px line-height.
