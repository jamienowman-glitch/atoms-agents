## Default values and ranges

- `typography_theme_id`: `roboto_flex_default`
- Default axes (desktop):
  - h1 wght 700 size 34px line_height 40px opsz 34 tracking 0
  - h2 wght 650 size 30px line_height 36px opsz 30 tracking 0
  - h3 wght 600 size 26px line_height 32px opsz 26 tracking 0
  - h4 wght 600 size 22px line_height 28px opsz 22 tracking 0
  - h5 wght 600 size 20px line_height 26px opsz 20 tracking 0
  - h6 wght 600 size 18px line_height 24px opsz 18 tracking 0
  - body_lg wght 450 size 18px line_height 26px opsz 18 tracking 0
  - body_md wght 450 size 16px line_height 24px opsz 16 tracking 0
  - body_sm wght 450 size 14px line_height 20px opsz 14 tracking 0
  - label/meta wght 550 size 13px line_height 18px opsz 13 tracking 10
  - button wght 600 size 15px line_height 20px opsz 15 tracking 20
  - caption/overline wght 500 size 12px line_height 16px opsz 12 tracking 30 (overline uppercase optional)
- Breakpoint scaling: reduce sizes 10–20% on mobile; modest on tablet.
- Allowed ranges: body text 14–18px, headings within ±6px of defaults; tracking deltas ±40; line-height multipliers 1.2–1.6.
- Fallback font stack: system sans matching metrics.
