## Icons registry

- Optional icons for preset selectors (e.g., heading/link/body glyphs).
- Tokens:
  - `typo_icon_size`
  - `typo_icon_colour` (paired with scheme text tokens)
  - `typo_icon_alignment` (start of row or inside pill)
- If UI is text-only, mark [NO ICONS] but keep tokens dormant for future swaps.
