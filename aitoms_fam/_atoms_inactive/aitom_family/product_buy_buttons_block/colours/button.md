# Buy Button Colours

## Tokens
| Role | Theme Token | Notes |
| :--- | :--- | :--- |
| **Primary Fill** | `button_primary_fill` | Inherits from theme scheme. |
| **Primary Text** | `button_primary_text` | Contrast ensured. |
| **Secondary Fill** | `button_secondary_fill` | Often transparent/outline. |
| **Secondary Text** | `button_secondary_text` | Inherits text_primary usually. |
| **Disabled Fill** | `surface_disabled` | Low opacity. |
