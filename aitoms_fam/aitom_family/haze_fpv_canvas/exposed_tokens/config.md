# Haze FPV Canvas - Exposed Tokens

These tokens control the runtime feel of the explorer.

| Token Key | Default Value | Description |
| :--- | :--- | :--- |
| `fpv_movement_speed` | `10.0` | Base movement speed (units/sec). |
| `fpv_run_multiplier` | `2.5` | Multiplier when Shift is held. |
| `fpv_look_sensitivity` | `0.002` | Radians per pixel of mouse movement. |
| `fpv_smoothing` | `0.1` | Lerp factor for camera ease (0.0 - 1.0). |
| `landscape_grid_scale` | `5.0` | Size of one grid cell in world units. |
| `landscape_max_height` | `100.0` | Physical height of the highest peak (height_score=1.0). |
| `node_visibility_range` | `500.0` | Distance at which nodes become visible. |
