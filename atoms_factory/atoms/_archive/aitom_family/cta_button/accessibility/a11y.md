# Accessibility Spec

## Roles & States
- **Role**: `button`, `link`, or specific widget role.
- **States**: `aria-expanded`, `aria-pressed`, `aria-disabled`.
- **Focus**: Must be focusable via Keyboard (Tab).
- **Label**: Ensure `aria-label` or visible label is present.

## Keyboard
- `Enter` / `Space`: Activate.
- `Tab`: Navigate in/out.
