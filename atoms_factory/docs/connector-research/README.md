# Connector Research

This directory stores raw research data for potential connectors. 

**purpose**: To provide a centralized seed location for connector requirement gathering.
**format**: TSV (Tab Separated Values).

## How to add
1. Add raw `.tsv` files to `tsv/`.
2. Add a snapshot markdown file to `snapshots/` (first 10 rows).
3. Update `INDEX.md`.

**Note**: These files are for research and planning only. They do not affect the runtime or the build process.
