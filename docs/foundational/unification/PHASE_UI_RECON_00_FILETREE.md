# UI Repo File Tree (Recon)

- apps/studio
  - src/App.tsx — studio shell wiring builder, inspector, analytics, SEO panel.
  - src/main.tsx — Vite entry.
  - src/components/CanvasFrame.tsx, SeoPanel.tsx
  - src/hooks/analytics.ts, src/seoClient.ts
- packages/transport/src
  - index.ts — CanvasTransport (HTTP/SSE/WS), safety handling, cursor cache.
  - identity_headers.ts — builds auth/context headers.
- packages/canvas-kernel/src
  - index.ts — CanvasKernel class re-export; reducer.ts initial state + op apply.
- packages/builder-core/src
  - index.ts — useBuilder hook wiring CanvasTransport + CanvasKernel + RegistryService.
- packages/builder-registry/src
  - models.ts — ComponentSchema, setting types, local SCHEMAS/SECTION_TEMPLATES.
  - service.ts — RegistryService fetches /registry/components.
  - index.ts — exports models + service.
- packages/ui-atoms/src
  - index.tsx — AtomRegistry (hero/text/button/image/box) + component exports.
- packages/projections/src
  - index.tsx — CanvasView + AtomRenderer composition layer with safety overlay.
- packages/builder-layout/src
  - Sidebar.tsx, components.tsx, index.ts — layout primitives for builder shell.
- packages/builder-copy/src
  - index.tsx — HeadlineBlock, TextBlock, ButtonBlock atoms.
- packages/builder-inspector/src
  - index.tsx — Inspector panel (prop editing, agent simulation hooks).
- packages/analytics/src
  - client.ts, index.ts — AnalyticsClient emitting pageview/cta events.
- packages/contracts/src
  - index.ts, types.ts — command/stream/event types used across packages.
- packages/agent-driver/src
  - index.ts, scripted agent utilities for simulated typing.
- packages/render-web/src
  - index.ts, App.tsx — render-web surface.
