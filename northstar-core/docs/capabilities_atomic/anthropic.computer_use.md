# Anthropic Computer Use
**ID:** `anthropic.computer_use`

## Usage
Controls a computer interface. Requires `anthropic-beta` header.

## Parameters
- `display_width`: Screen width px.
- `display_height`: Screen height px.
- `display_number`: Monitor index.
